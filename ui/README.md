# ui/

## Purpose

UI/UX design system and component library.

## Ownership

**UI Designer Agent** has primary ownership of this directory.

## Contents

- **design-tokens/**: Colors, typography, spacing
- **components/**: Reusable UI components
- **patterns/**: Common UI patterns
- **assets/**: Icons, images, fonts

## Structure

```
ui/
├── design-tokens/
│   ├── colors.json
│   ├── typography.json
│   └── spacing.json
├── components/
│   ├── Button/
│   ├── Input/
│   └── Modal/
├── patterns/
│   ├── forms/
│   └── navigation/
└── assets/
    ├── icons/
    └── images/
```

## Design Principles

- Consistency across applications
- Accessibility-first design
- Responsive and adaptive
- Clear visual hierarchy

## Usage

Components from this library are consumed by the frontend service to ensure consistent UI across applications.
