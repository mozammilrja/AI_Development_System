# Platform Simulations

## Purpose

Houses test simulation suites used by the **Tester** agent for end-to-end validation, performance benchmarking, resilience testing, and automated bug reproduction.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `browser_tests/` | Playwright-based end-to-end browser test suites |
| `load_testing/` | Performance and load test scenarios |
| `chaos_testing/` | Resilience and failure-injection test configs |
| `bug_simulation/` | Automated bug reproduction environments |

## Owner

**Tester** agent owns test implementations. **QA** agent defines test strategy and acceptance criteria.
