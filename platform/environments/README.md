# Platform Environments

## Purpose

Provides isolated development and testing environments for agents, including mock servers, browser automation contexts, code sandboxes, and test simulation configs.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `api_mock_server/` | Stubbed API endpoints for offline agent testing |
| `browser_env/` | Browser automation environment configs (Playwright contexts) |
| `code_sandbox/` | Isolated sandbox for agent-generated code execution |
| `test_simulation/` | Simulated test scenarios and fixture data |

## Owner

**DevOps** agent manages environment configuration.
