# services/frontend/

## Purpose

Frontend web applications built by the AI system.

## Ownership

**Frontend Engineer Agent** has exclusive write access to this directory.

## Structure

```
frontend/
├── src/
│   ├── components/   # React components
│   ├── pages/        # Page components
│   ├── hooks/        # Custom hooks
│   ├── stores/       # State management
│   ├── services/     # API clients
│   └── utils/        # Utilities
├── tests/
│   ├── unit/
│   └── e2e/
├── public/
├── package.json
└── tsconfig.json
```

## Tech Stack

- **Framework**: React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State**: Zustand or similar
- **Testing**: Jest + Playwright

## UI Guidelines

- Functional components with hooks
- Responsive design
- Accessibility (WCAG 2.1)
- Component documentation
