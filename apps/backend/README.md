# Apps Backend

This workspace contains the **backend services** for the primary application.

## Stack

- **Runtime**: Node.js + TypeScript
- **Framework**: Express.js or Fastify (to be decided per feature)
- **Database**: PostgreSQL (via Docker Compose)
- **Cache**: Redis

## Getting Started

```bash
cd apps/backend
npm install
npm run dev
```

## Owner

**Backend Agent** — all files in this directory are owned by the Backend agent.
