# .claude/

## Purpose

Claude Code configuration and customization files.

## Contents

- **commands/**: Custom slash commands for the AI system
- **skills/**: Domain-specific knowledge and instructions
- **settings.json**: Claude Code workspace settings
- **project_context.md**: Project context for AI interactions

## Structure

```
.claude/
├── commands/         # Custom /commands for workflows
├── skills/           # Specialized knowledge files
├── settings.json     # Workspace configuration
└── project_context.md # Context for AI interactions
```

## Usage

Files in this directory customize Claude Code behavior for this specific project, enabling specialized workflows and commands.
