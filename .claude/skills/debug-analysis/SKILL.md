# Debug Analysis Skill

## Description
Advanced debugging skill for systematic bug analysis, root cause identification, and fix generation.

## Capabilities
- Parse and analyze error messages and stack traces
- Search codebase for related code patterns
- Identify root cause through causal analysis
- Generate fix candidates ranked by confidence
- Validate fixes in sandbox environment
- Detect regression risks from proposed fixes
- Analyze memory leaks and performance issues

## Analysis Pipeline
1. **Symptom Collection**: Gather error logs, stack traces, reproduction steps
2. **Hypothesis Generation**: Generate potential causes based on error patterns
3. **Code Analysis**: Trace execution paths and data flows
4. **Root Cause Isolation**: Narrow down to specific code location
5. **Fix Generation**: Produce ranked fix candidates
6. **Validation**: Test fixes against regression suite

## Supported Bug Types
- Runtime exceptions and crashes
- Logic errors and incorrect output
- Performance degradation
- Memory leaks
- Race conditions and concurrency bugs
- Integration failures
- UI rendering issues
