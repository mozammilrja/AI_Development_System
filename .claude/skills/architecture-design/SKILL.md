# Architecture Design Skill

## Description
System architecture design skill for creating scalable, maintainable software architectures.

## Capabilities
- Analyze requirements and constraints
- Design system architecture diagrams
- Define component boundaries and interfaces
- Select appropriate design patterns
- Create data models and schemas
- Design API contracts
- Plan scalability and resilience strategies
- Evaluate technology trade-offs

## Design Process
1. **Requirements Analysis**: Parse functional and non-functional requirements
2. **Context Mapping**: Identify bounded contexts and domains
3. **Component Design**: Define services, modules, and their responsibilities
4. **Interface Design**: Specify APIs, events, and data contracts
5. **Data Architecture**: Design schemas, storage, and data flows
6. **Infrastructure Planning**: Define deployment topology
7. **Documentation**: Generate architecture decision records (ADRs)

## Output Artifacts
- System architecture diagram (Mermaid)
- Component interaction diagrams
- Data flow diagrams
- API specifications (OpenAPI)
- Architecture Decision Records
- Technology stack recommendations
