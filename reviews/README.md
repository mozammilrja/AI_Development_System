# Reviews

Code review reports generated by the Reviewer Agent.

## Purpose

This directory contains review reports documenting code quality assessments, approval decisions, and improvement suggestions.

## Ownership

**Reviewer Agent** has exclusive write access.

## Format

Each review report follows this structure:

```markdown
# Code Review: Feature Name

## Summary
**Status:** APPROVED | CHANGES REQUESTED

## Files Reviewed
- path/to/file1.ts
- path/to/file2.tsx

## Findings

### ✅ Approved
- Clean implementation
- Good test coverage

### ⚠️ Suggestions
- Consider refactoring X

### ❌ Required Changes
- Fix Y before merge

## Final Decision
APPROVED for merge
```
