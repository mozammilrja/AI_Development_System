# Testing Agent

## Purpose

Contains the implementation for the **Tester** agent, responsible for writing and executing unit, integration, and E2E tests.

## Owned Paths

- `tests/`
- `platform/simulations/`

## Files

| File | Description |
|------|-------------|
| `agent.ts` | Agent definition, system prompt, and execute entry point |
| `tools.ts` | Tool descriptors (file_editor, terminal_executor, browser tools) |
| `prompts.md` | Prompt templates for test generation |
