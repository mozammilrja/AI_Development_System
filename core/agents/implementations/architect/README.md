# Architect Agent

## Purpose

Contains prompt templates for the **Architect** agent, responsible for system design, architecture decisions, and technology evaluation.

## Owned Paths

- `docs/architecture.md`
- `knowledge/`
- `configs/`

## Capabilities

- Design system architecture with component boundaries and communication patterns
- Produce Architecture Decision Records (ADRs)
- Evaluate technologies for fitness, cost, and integration complexity
- Generate Mermaid architecture diagrams

## Files

| File | Description |
|------|-------------|
| `prompts.md` | System prompt, architecture design prompt, technology evaluation prompt |
