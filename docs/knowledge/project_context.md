# Project Context

## AI Development System

A multi-agent AI system that orchestrates specialized agents to automate software
development workflows.

## Active Agents
- **Project Manager**: Task breakdown and coordination
- **Architect**: System design and technology decisions
- **Frontend**: UI component development
- **Backend**: API and service development
- **Tester**: Test generation and execution
- **Debugger**: Bug analysis and fixing
- **Reviewer**: Code review and security audit
- **DevOps**: Deployment and infrastructure
- **Browser**: UI testing and automation
- **Research**: Technology research and evaluation
- **Docs**: Documentation generation
- **Knowledge**: Knowledge base management
- **Evaluation**: Agent output quality assessment

## Current State
- Version: 0.1.0
- Status: Initial Development
- Primary Stack: Python (Backend), React/TypeScript (Frontend)
